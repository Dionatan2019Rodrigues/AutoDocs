# AutoDocs
Sistema desenvolvido para automatizar a geração de documentos institucionais relacionados a projetos e convênios da UFSM. A plataforma centraliza o preenchimento de informações administrativas, financeiras e operacionais por meio de formulários, gerando automaticamente documentos em PDF padronizados conforme os modelos exigidos pela Coordenadoria de Projetos.

### Como executar ? 

##### Baserow
```
npm run baserow
```
##### Formbricks
```
npm run formbricks
```
##### Typebot
```
npm run typebot
```
